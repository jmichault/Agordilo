Description :
     Accordeur is a software designed to help you to tune your music instrument.

    Accordeur is derived from Dan's Tuner.
    Essentials differences are :
  • Display analysed spectrum.
  • For grave pitches, changes number of samples, so very grave notes are well analysed.
  • Fine compute of the pitch is made by searching nearest parabole, not by cubic interpolation.
