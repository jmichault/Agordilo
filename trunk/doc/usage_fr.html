Description :
     Accordeur est un logiciel permettant d'accorder un instrument de musique à l'aide d'un ordinateur.

Historique :
      Voulant accorder moi-même un piano, j'ai cherché des logiciels pour me permettre de le faire. J'en ai trouvé quelques uns, mais aucun ne m'a permis de tout faire : la plupart marchait bien pour les notes moyennes, très peu pour les notes aigues, et aucun pour les notes très graves.
    J'ai donc décidé d'en choisir un délivré en licence GPL, et d'y apporter des modifications pour arriver à accorder mon piano. J'ai choisi de partir de Dan's Tuner.
    Les principales améliorations apportées sont :
  • Affichage du spectre analysé.
  • Augmentation du nombre d'échantillons pris en compte dans le calcul pour les notes graves.
  • Calcul de la note exacte par recherche de la parabole la plus proche du pic, au lieu d'utiliser le polynôme d'ordre 3 passant par les 4 points les plus proches du pic.



Utilisation :
    Lancer le programme :
       ./accordeur
        • Cliquer sur le bouton "Calibrer le bruit". Ceci permet de mieux reconnaître les notes en ne tenant compte que des sons supérieurs au bruit de fond analysé à cette étape.
        • choisir la note qu'on veut accorder.
        • La jouer sur l'instrument.
        • accorder l'instrument de manière à ce que le pic principal soit aligné sur le trait vert. Le premier trait rouge à droite du trait vert indique la note immédiatement supérieure. Le premier trait rouge à gauche indique la note immédiatement inférieure. Les autres traits rouge plus à gauche indiquent les harmoniques 1/3, 1/5, 1/7, ... Ils permettent d'affiner encore plus le réglage.
  • Enfin affiner l'accord en rapprochant le plus possible la fréquence affichée de la fréquence de la note désirée.

Installation :
    Seule une version linux est disponible sur sourceforge.
    extraire l'exécutable où bon vous semble :
       tar zxf  accordeur*.tgz

Compilation :
     Vous avez besoin du compilateur gcc et de qt3-designer.
  • extraire les sources :
             tar zxf accordeur-x.x.x-src.tar.gz
  • compiler la librairie portaudio :
            cd accordeur/lib/portaudio
            make
  • compiler accordeur :
            cd ../..
            make
