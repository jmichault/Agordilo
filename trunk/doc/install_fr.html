Installation		 Installation :
    Seules une version linux et une version windows sont disponibles sur sourceforge.
  Linux :
    extraire l'exécutable où bon vous semble :
       tar zxf  accordeur*.tgz
       cd accordeur
       ./install
  Windows :
     pas d'installeur pour l'instant.
     récupérer le .zip ou le .7z, et extraire le répertoire accordeur où bon vous semble, puis double-cliquer sur accordeur.exe.

Compilation :
     Vous avez besoin du compilateur gcc et de qt3-designer.
  • extraire les sources :
             tar zxf accordeur-x.x.x-src.tar.gz
  • compiler la librairie portaudio :
            cd accordeur/lib/portaudio
            make
  • compiler accordeur :
            cd ../..
            make
