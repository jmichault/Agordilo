Installation		 Installation :
    Seules une version linux et une version windows sont disponibles sur sourceforge.
  Linux :
    extraire l'exécutable où bon vous semble :
       rpm -i accordeur-x.x.x-x.i586.rpm

  Windows :
     pas d'installeur pour l'instant.
     récupérer le .7z, et extraire le répertoire accordeur où bon vous semble, puis double-cliquer sur accordeur.exe.

Compilation :
     Vous avez besoin du compilateur gcc et de qt4-designer.
  • extraire les sources :
             tar zxf accordeur-x.x.x-src.tar.gz
  • compiler accordeur sous linux :
            cd accordeur
            qmake
            make
