Installation
Installation :
    Seule une version linux est disponible sur sourceforge.
    extraire l'exécutable où bon vous semble :
       tar zxf  accordeur*.tgz
       cd accordeur
       ./install
Compilation :
     Vous avez besoin du compilateur gcc et de qt3-designer.
  • extraire les sources :
             tar zxf accordeur-x.x.x-src.tar.gz
  • compiler la librairie portaudio :
            cd accordeur/lib/portaudio
            make
  • compiler accordeur :
            cd ../..
            make
