Install
How to install :
    Only linux binary is available on sourceforge.
       tar zxf  accordeur*.tgz
       cd accordeur
       ./install
How to compile :
     You need gcc and qt3-designer.
  • extract sources :
             tar zxf accordeur-x.x.x-src.tar.gz
  • compile portaudio library :
            cd accordeur/lib/portaudio
            make
  • compile accordeur :
            cd ../..
            make
